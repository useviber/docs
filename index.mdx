## ğŸ—ï¸ How It Works

1. **Install Viber CLI**: `npm install -g @useviber/cli`
2. **Install Cursor CLI**: `npm install -g @cursor/cli` (for Cursor Bridge)
3. **Start Viber** in your project directory: `viber start`
4. **Viber creates a proxy** that intercepts your running app
5. **Your app opens automatically** in the browser with Viber injected
6. **Click the floating Viber icon** (ğŸ¤–) in the bottom-right corner
7. **Select Cursor Bridge** as your AI provider
8. **Select any element** on your page
9. **Describe what you want to change** using natural language
10. **Watch the magic happen!** âœ¨

**No manual integration required!** Viber automatically detects your framework and injects itself.
