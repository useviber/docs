## What are Viber Extensions?

**Viber Extensions** are IDE-specific integrations that connect your development environment directly to Viber's AI-powered workflow. They enable seamless communication between your IDE's AI tools and Viber's element selection system.

## 🔄 How the Extension Workflow Works

### 1. Install & Setup
- **Install Viber Extension** in your IDE (VS Code, Cursor, etc.)
- **Start Viber** in your frontend project: `viber start`
- **Extension auto-connects** to the running Viber instance

### 2. Select AI Provider
- **Open Viber's UI** in your browser
- **Choose "IDE AI"** as your AI provider
- **Select your extension** from the available IDE extensions

### 3. Start Using AI
- **Select elements** on your page using Viber's interface
- **Send prompts** directly to your IDE's AI assistant
- **Get real-time responses** and code changes

## 🚀 Supported Environments

### Primary Support

- **VS Code** - Full extension with marketplace installation
- **Cursor** - Built-in integration with AI Bridge
- **Local AI Tools** - Cursor CLI and other CLI-based AI agents

### Extended Support

- **Any CLI-based AI tool** - Via Viber's AI Bridge system
- **Browser extensions** - Chrome, Firefox, Safari
- **Desktop apps** - Electron-based applications

## 🔌 VS Code Extension

### Installation

1. **Open VS Code**
2. **Go to Extensions** (Ctrl+Shift+X / Cmd+Shift+X)
3. **Search for "Viber"**
4. **Click Install**

Or install via command line:

```bash
code --install-extension useviber.viber
```

### Configuration

Create `.vscode/settings.json` in your project:

```json
{
    "viber.enabled": true,
    "viber.autoStart": true,
    "viber.workspace": "/path/to/your/project",
    "viber.debug": false,
    "viber.theme": "auto"
}
```

### How It Works

1. **Install Viber Extension** in VS Code
2. **Start Viber** in your frontend project directory: `viber start`
3. **Extension auto-connects** to the running Viber instance
4. **In Viber's UI**, select "IDE AI" as your AI provider
5. **Choose your extension** from the available IDE extensions
6. **Start sending prompts** directly to your IDE's AI assistant

### Features

- **Automatic connection** to Viber when you start your project
- **IDE AI integration** - Connect to your IDE's built-in AI tools
- **Extension selection** - Choose which IDE extension to use
- **Direct communication** - Send prompts straight to your IDE's AI
- **Real-time updates** - See changes as they're applied

## 🖥️ Desktop Applications

### Electron Apps

```javascript
// main.js
const { app, BrowserWindow } = require('electron');
const { Viber } = require('@useviber/plugin');

app.whenReady().then(() => {
    const win = new BrowserWindow({
        width: 1200,
        height: 800,
        webPreferences: {
            nodeIntegration: true,
            contextIsolation: false
        }
    });

    win.loadFile('index.html');

    // Initialize Viber in Electron
    Viber.init({
        apiKey: process.env.VIBER_API_KEY,
        workspace: process.cwd(),
        isElectron: true
    });
});
```

### Tauri Apps

```rust
// main.rs
use tauri::Manager;

fn main() {
    tauri::Builder::default()
        .setup(|app| {
            let window = app.get_window("main").unwrap();
            
            // Initialize Viber in Tauri
            window.eval(&format!(
                r#"
                import('@useviber/plugin').then(({{ Viber }}) => {{
                    Viber.init({{
                        apiKey: '{}',
                        workspace: '{}',
                        isTauri: true
                    }});
                }});
                "#,
                std::env::var("VIBER_API_KEY").unwrap_or_default(),
                std::env::current_dir().unwrap().to_str().unwrap()
            )).unwrap();
            
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

## 🔧 Extension Configuration

### Global Settings

```json
{
    "viber": {
        "enabled": true,
        "autoConnect": true,
        "defaultWorkspace": "/path/to/default/project",
        "theme": "auto",
        "debug": false,
        "notifications": true
    }
}
```

### Project-specific Settings

```json
{
    "viber.workspace": "/path/to/this/project",
    "viber.framework": "react",
    "viber.port": 3000,
    "viber.viberPort": 8426
}
```

### Environment Variables

```bash
# .env file
VIBER_EXTENSION_ENABLED=true
VIBER_AUTO_CONNECT=true
VIBER_DEFAULT_WORKSPACE=/path/to/project
VIBER_DEBUG=false
```

## 🔍 Troubleshooting

### Common Issues

**Extension not connecting:**
```bash
# Check if Viber is running
viber status

# Restart Viber
viber start

# Check extension status in IDE
# VS Code: View -> Output -> Viber
```

**IDE AI provider not working:**
```bash
# Verify extension is installed and enabled
# Check Viber's AI Provider Selection for "IDE AI" option
# Ensure your IDE extension is selected in the dropdown
```

**Element selection not working:**
- Ensure Viber is running in your project
- Check browser console for errors
- Verify extension is enabled
- Restart your IDE

### Debug Mode

```json
{
    "viber.debug": true,
    "viber.verbose": true,
    "viber.logLevel": "debug"
}
```

### Extension Logs

**VS Code:**
- View → Output → Viber
- Developer Tools → Console

**Cursor:**
- Help → Toggle Developer Tools → Console

**Browser Extensions:**
- Right-click extension → Inspect
- Check browser console

## 📱 Mobile Development

### React Native

```javascript
// App.js
import { Viber } from '@useviber/plugin';

const App = () => {
    useEffect(() => {
        Viber.init({
            apiKey: 'your-api-key',
            workspace: '/path/to/project',
            isReactNative: true,
            mobile: {
                position: 'bottom-center',
                touchFriendly: true
            }
        });
    }, []);

    return (
        <View style={styles.container}>
            <Text>My React Native App</Text>
        </View>
    );
};
```

### Flutter

```dart
// main.dart
import 'package:flutter/material.dart';
import 'package:viber_plugin/viber_plugin.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // Initialize Viber
    ViberPlugin.init(
      apiKey: 'your-api-key',
      workspace: '/path/to/project',
    );
    
    return MaterialApp(
      title: 'Flutter App',
      home: MyHomePage(),
    );
  }
}
```
