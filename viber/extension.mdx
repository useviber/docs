## ðŸ”Œ VS Code Extension

### Installation

1. **Open VS Code**
2. **Go to Extensions** (Ctrl+Shift+X / Cmd+Shift+X)
3. **Search for "Viber"**
4. **Click Install**

Or install via command line:

```bash
code --install-extension useviber.viber
```

### Configuration

Create `.vscode/settings.json` in your project:

```json
{
    "viber.enabled": true,
    "viber.autoStart": true,
    "viber.workspace": "/path/to/your/project",
    "viber.debug": false,
    "viber.theme": "auto"
}
```

### Usage

1. **Open your project** in VS Code
2. **Start Viber** in your project directory: `viber start`
3. **Extension auto-connects** to the running Viber instance
4. **Use Command Palette** (Ctrl+Shift+P / Cmd+Shift+P):
   - `Viber: Open Interface`
   - `Viber: Select Element`
   - `Viber: Apply Changes`

### Features

- **Command Palette integration** - Quick access to Viber commands
- **Status bar indicator** - Shows connection status
- **Output panel** - View Viber logs and errors
- **Workspace management** - Auto-detects project structure

## ðŸŽ¯ Cursor Integration

### Built-in Support

Cursor has native CLI integration, making Viber AI Bridge integration seamless:

1. **Install Viber CLI**: `npm install -g @useviber/cli`
2. **Install Cursor CLI**: `npm install -g @cursor/cli`
3. **Start Viber** in your project: `viber start`
4. **Select Cursor Bridge** in the AI Provider Selection

### AI Bridge Configuration

The AI Bridge automatically detects and uses Cursor CLI:

```bash
# Verify Cursor CLI installation
cursor-agent --version

# Start Viber with AI Bridge
viber start
```

### Usage in Cursor

- **Select Cursor Bridge** as your AI provider in Viber
- **Element selection** provides context to Cursor CLI
- **Local processing** - No external servers required
- **Real-time streaming** - See AI responses as they generate

## ðŸ–¥ï¸ Desktop Applications

### Electron Apps

```javascript
// main.js
const { app, BrowserWindow } = require('electron');
const { Viber } = require('@useviber/plugin');

app.whenReady().then(() => {
    const win = new BrowserWindow({
        width: 1200,
        height: 800,
        webPreferences: {
            nodeIntegration: true,
            contextIsolation: false
        }
    });

    win.loadFile('index.html');

    // Initialize Viber in Electron
    Viber.init({
        apiKey: process.env.VIBER_API_KEY,
        workspace: process.cwd(),
        isElectron: true
    });
});
```

### Tauri Apps

```rust
// main.rs
use tauri::Manager;

fn main() {
    tauri::Builder::default()
        .setup(|app| {
            let window = app.get_window("main").unwrap();
            
            // Initialize Viber in Tauri
            window.eval(&format!(
                r#"
                import('@useviber/plugin').then(({{ Viber }}) => {{
                    Viber.init({{
                        apiKey: '{}',
                        workspace: '{}',
                        isTauri: true
                    }});
                }});
                "#,
                std::env::var("VIBER_API_KEY").unwrap_or_default(),
                std::env::current_dir().unwrap().to_str().unwrap()
            )).unwrap();
            
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

## ðŸ” Troubleshooting

### Common Issues

**Extension not connecting:**
```bash
# Check if Viber CLI is running
viber status

# Restart Viber
viber restart

# Check extension logs
# VS Code: View -> Output -> Viber
```

**AI Bridge connection failed:**
```bash
# Check AI Bridge status
viber bridge:status

# Verify Cursor CLI installation
cursor-agent --version

# Check API key
viber whoami
```

**Element selection not working:**
- Ensure Viber is running in your project
- Check browser console for errors
- Verify extension is enabled
- Restart your IDE
