## Installation

```bash
npm install -g @useviber/cli
```

<Tip>
The CLI automatically installs the MCP server on postinstall. If you encounter permission errors, run `viber mcp:install` manually.
</Tip>

## üöÄ Core Commands

### Start Viber

Start Viber in your current project directory:

```bash
viber start
```

**What happens:**
- Auto-detects your framework and ports
- Starts a proxy server for your app
- Injects the Viber widget
- Connects to the communication server

**Options:**
```bash
# Custom app port
viber start --port 8080

# Custom Viber CLI port
viber start --viber-port 9000

# Force specific framework
viber start --framework react

# Enable debug logging
viber start --debug

# Use CDN for plugin (production)
viber start --use-cdn

# Custom plugin CDN URL
viber start --plugin-cdn-url https://cdn.example.com/viber-plugin.js
```

### Authentication

```bash
# Login to Viber
viber login

# Check authentication status
viber whoami

# Logout
viber logout

# Check overall status
viber status
```

## üîå MCP Commands

### Install MCP Server

```bash
# Install MCP globally
viber mcp:install

# Check MCP installation status
viber mcp:status
```

### MCP Management

```bash
# Start MCP server manually
viber mcp:start

# Stop MCP server
viber mcp:stop

# Restart MCP server
viber mcp:restart
```

## üéØ Framework Detection

Viber automatically detects your framework, but you can override it:

```bash
# Available frameworks
viber start --framework react     # React/Next.js
viber start --framework vue       # Vue/Nuxt
viber start --framework angular   # Angular
viber start --framework svelte    # Svelte/SvelteKit
viber start --framework vanilla   # Vanilla HTML/CSS/JS
```

## üîß Configuration

### Environment Variables

```bash
# .env file
LOCAL=1                    # Use localhost endpoints
VIBER_DEBUG=true          # Enable verbose logging
VIBER_PORT=8426          # Custom CLI port
VIBER_APP_PORT=3000      # Custom app port
```

### Auto-generated Configuration

Viber creates a `viber.json` file in your project:

```json
{
  "appPort": 3000,
  "viberPort": 8426,
  "framework": "react",
  "workspace": "/path/to/your/project",
  "pluginUrl": "http://localhost:8426/viber-plugin.js"
}
```

## üåê Network & Ports

### Port Configuration

```bash
# Default ports
App Port: 3000 (React), 5173 (Vue), 4200 (Angular)
Viber Port: 8426
MCP Port: 3337

# Custom ports
viber start --port 8080 --viber-port 9000
```

### Network Options

```bash
# Bind to specific host
viber start --host 0.0.0.0

# Use HTTPS
viber start --https

# Custom SSL certificate
viber start --ssl-cert ./cert.pem --ssl-key ./key.pem
```

## üö® Common Issues

### Port Conflicts

```bash
# Kill process using port
lsof -ti:3000 | xargs kill -9

# Use different port
viber start --port 3001
```

### Permission Issues

```bash
# Fix MCP installation
viber mcp:install

# Check permissions
ls -la ~/.viber/
```

### Connection Issues

```bash
# Check network
viber network:test

# Reset configuration
viber config:reset
```
